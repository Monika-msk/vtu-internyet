Subscribe to Daily VTU Internship Alerts

Get a single email every day at 9:00 AM IST with new internships posted on VTU Internyet. Your email will only be used for sending these alerts.

Step 1 — Open the subscription form

Open Subscription Form

If the button is disabled, the admin hasn't linked the form yet.

How this works
  1. You submit your email via a Google Form.
  2. Your email is saved in a Google Sheet managed by the project admin.
  3. The system fetches the sheet as a read-only CSV and sends alerts every morning.
  4. You can unsubscribe anytime via the link in the email footer (coming soon).
For Admins
  1. Create a Google Form with one field: Email (response validation: email).
  2. Connect it to a Google Sheet (Responses → Link to Sheets).
  3. In the Sheet: File → Share → Publish to the web → select the responses sheet → format: Comma-separated values (.csv) → Publish.
  4. Copy the public CSV link and save it as a GitHub secret: SUBSCRIBERS_CSV_URL.
  5. Also set sender credentials: SENDER_EMAIL, SENDER_PASSWORD (Gmail App Password) in repo secrets.
  6. Optional: Put your Google Form link below so the Subscribe button works.

GitHub Actions runs this job daily at 03:30 UTC (09:00 IST). See .github/workflows/internship-watcher.yml.

Configure Subscribe button

Replace the placeholder below by editing this file and inserting your public Google Form link:

<script>
// Set your Google Form link here
const GOOGLE_FORM_LINK = ""; // e.g., "https://docs.google.com/forms/d/e/.../viewform"
if (GOOGLE_FORM_LINK) {
  const a = document.getElementById('formLink');
  a.href = GOOGLE_FORM_LINK;
} else {
  document.getElementById('formLink').classList.add('muted');
  document.getElementById('formLink').style.pointerEvents = 'none';
}
</script>
